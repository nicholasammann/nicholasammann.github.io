About Me

I'm a software engineer currently pursuing a Bachelor of Science in Computer Science and Real-Time Interactive Simulation at DigiPen Institute of Technology. I've worked on academic project teams of 8-12 people creating video games in custom game engines written in C++.

Most of my project experience is in tools programming, although I have some knowledge of core engine and graphics programming. I enjoy diving into a codebase and figuring out how to write tools that work with the existing code. Currently I am working on an editor for a custom 3D game engine called YTE.

Lately, I have taken an interest in UI/UX design. I am developing a GUI for Perforce, targeted towards game developers that might benefit from a directed source control workflow.
